# Learning

A personal repository documenting hands-on learning across computer science and mathematics topics.

## Structure

```
Learning/
├── computer-science/
│   └── cpp/
│       ├── data-structures/
│       │   ├── foundation-challenges/   # Core DS exercises
│       │   └── binary-search-tree/      # BST implementation
│       └── game-dev/
│           ├── sfml/
│           │   ├── ping/                # Ping game (SFML + CMake)
│           │   └── phys-simulator/      # Ball physics simulation (SFML + CMake)
│           └── raylib/
│               └── basic-app/           # Raylib starter project
└── math/
    └── linear-algebra/
    └── discrete-structures/
        └── cryptography/
            ├── diffie-hellman/          # Diffie-Hellman key exchange (C++ & Python)
            └── el-gamal/                # El-Gamal asymmetric encryption
```

## Topics

### Computer Science

**Data Structures** — Foundation challenges and a binary search tree built in C++.

**cpp Libraries** — 2D projects using SFML (ping game, physics simulator) and a Raylib starter. All C++ projects use CMake.

### Mathematics

**Cryptography** — Implementations of classic asymmetric protocols: Diffie-Hellman key exchange and El-Gamal encryption, written in C++ and Python.

**Linear Algebra** — Notes and exercises.
